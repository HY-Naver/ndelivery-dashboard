📦 온보딩 담당자 뷰 — 물류사 배정 입력 방법

커솔마 등록 완료 직후, 풀필먼트팀에서 적합한 물류사를 선정하여 직접 입력합니다.

물류사 배정 절차

  1. 온보딩 담당자 뷰 탭 접속
  2. 미배정 업체의 + 물류사 배정 버튼 클릭
  3. 물류사 선택 (컬리 / 아르고 / CJ대한통운 / 품고 / 기타)
  4. 배정일은 오늘 날짜로 자동 입력됨 (수동 변경 가능)
  5. 메모 입력 후 저장 → 테이블에 즉시 반영
배정 완료된 업체는 초록색 물류사 뱃지로 표시되며 클릭하면 수정할 수 있습니다. 배정일은 수동으로 변경 가능합니다.

❓ 자주 묻는 질문

Q. 물류사 배정일이 틀렸는데 수정하려면?

온보딩 담당자 뷰에서 초록색 물류사 뱃지를 클릭하면 수정 모달이 열립니다. 날짜 필드를 직접 수정하고 저장하세요.

Q. 리드타임 기준일은 어떻게 되나요?
